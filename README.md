# AI-Blockchain-Online-Examination-System
## Repository Structure
- frontend/: User interface for examination workflow
- backend/: API routes, services, and system logic
- ai-models/: Face recognition and object detection modules
- blockchain/: Blockchain logging and hash management

## Note on Models
Large pretrained model files are excluded from this repository. The system uses FaceNet-based facial embeddings and YOLOv8n for object detection.
